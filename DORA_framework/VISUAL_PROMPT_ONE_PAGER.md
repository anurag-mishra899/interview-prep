## DETAILED VISUAL SPECIFICATIONS

### Typography Hierarchy
- **Title**: 48pt, Bold, Sans-serif (e.g., Helvetica Neue Bold)
- **Section Headers**: 24pt, Bold, Sans-serif
- **Body Text**: 14pt, Regular, Sans-serif
- **Callouts/Labels**: 12pt, Medium, Sans-serif
- **Fine Print**: 10pt, Regular, Sans-serif

### Icon Style
- Line icons (not filled)
- Consistent stroke width (2px)
- Rounded corners
- Single color per icon matching section color
- Size: 32px x 32px for main icons, 24px for secondary

### Spacing & Alignment
- 20px padding around all major sections
- 16px gaps between elements
- All text left-aligned within boxes
- Centered alignment for icons above text
- Consistent margins: 40px from slide edges

### Visual Effects
- Subtle drop shadows on boxes (2px offset, 10% opacity)
- Gradient backgrounds where indicated (15-degree angle)
- Glow effect on "NEW" badges (5px blur, 30% opacity)
- Connecting lines: 2px dashed lines in light gray

## COMPOSITION GUIDELINES

### Visual Hierarchy (Eye Flow)
1. **First view**: Title bar (top)
2. **Second view**: Central 7-dimension diagram (largest, centered)
3. **Third view**: ROI numbers (right, amber color draws eye)
4. **Fourth view**: Problem/Solution narrative (left column)
5. **Fifth view**: Implementation timeline (right column)
6. **Sixth view**: Bottom differentiators (footer)

### Balance
- Left column (problem/solution): Text-heavy, vertical stacking
- Center: Visual diagram, minimal text
- Right column: Numbers/timeline, mixed text and graphics
- Bottom: Horizontal symmetry with 3 equal elements

### White Space Usage
- Generous breathing room around central diagram
- Tight packing in left column (conveys density of problems)
- Clean spacing in right column (conveys clarity of benefits)
- Minimum 10% of slide should be pure white space

## ACCESSIBILITY CONSIDERATIONS

### Text Contrast
- All text must have 4.5:1 contrast ratio minimum
- White text on dark blue: WCAG AAA compliant
- Dark text on light backgrounds: WCAG AA minimum

### Color Blindness
- Do not rely solely on color to convey information
- Use icons + color combination
- Red/green pairs include different shapes/icons
- Patterns/textures supplement color coding

### Readability
- Minimum font size: 10pt (readable from 10 feet)
- No more than 5 colors total
- Clear visual hierarchy
- Adequate white space between elements
